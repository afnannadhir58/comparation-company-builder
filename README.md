ai-comps-builder : Semantic Valuation Engine for Equity Research

Overview:
A Quantitative Equity Research tool that leverages Large Language Models (LLMs) to automate the most time-consuming part of financial modeling: finding the right comparable companies. Instead of relying on rigid industry classification codes, this engine uses Llama 3.3 to find peers based on actual business model mechanics.

Key Metrics & Features:

Sub-10 Second Execution: Generates a complete valuation comps table and implied Enterprise Value (EV) range in seconds.

100% LLM-Driven Rationale: Replaces human bias in peer selection with objective, semantic similarity scoring and provides a 1-sentence analytical rationale for every matched ticker.

Zero-Shot NLP Processing: Utilizes Anthropic/Groq endpoints to dissect unstructured longBusinessSummary data into structured JSON parameters without requiring model fine-tuning.

Real-Time Financial Analytics: Pipeline connected directly to Yahoo Finance for live Market Cap, Debt, Cash, and NTM/LTM EBITDA scraping.
